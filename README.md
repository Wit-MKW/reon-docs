# reon-docs

Various documentation &amp; code written as an aid to the REON project.
This includes:
- documentation of each game's server structure.
- scripts (mostly in Python) to generate DLC files.

...and more!

Before making this repository, I mostly posted such things to the REON Discord
server, but that made it difficult to make fixes: I'd either have to clutter a
given channel with multiple versions of the same file, or post errata as
separate messages, making them easy to miss (and also cluttering the channel
somewhat).
